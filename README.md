## 📖 Tentang Project
**SHERLOCK-TERMUX** adalah alat intelijen sumber terbuka (OSINT) yang dirancang khusus untuk lingkungan Termux. Alat ini menggabungkan kecepatan **Node.js** untuk scanning dan stabilitas **Python** untuk manajemen menu. Dengan tampilan UI yang mewah dan modern, alat ini memudahkan kamu melacak jejak digital berdasarkan username di berbagai platform global.

## 🚀 Fitur Utama
- ✨ **Ultra-Fast Scanning**: Menggunakan *Asynchronous Engine* untuk mengecek puluhan situs dalam hitungan detik.
- 💎 **Luxury Terminal UI**: Tampilan gradien neon dan tabel yang rapi.
- 🛠 **Hybrid System**: Menggabungkan Python dan JavaScript untuk performa terbaik.
- 📥 **Extra Tools**: Dilengkapi dengan TikTok Downloader (No Watermark).
- 🤫 **Silent Installer**: Proses instalasi bersih tanpa log yang berantakan.
